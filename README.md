## 找硬币问题：

店员给顾客找零钱，已知共需要找出n元钱，并且有k种硬币面值，问最少需要几枚硬币。

输入格式：第一行输入目标总零钱数n和硬币种类数k；第二行及以下k行输入每种硬币对应的面值。

输出格式：打印所需最少硬币数。
